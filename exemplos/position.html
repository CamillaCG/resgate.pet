A position sticky é usada para fixar elementos conforme acontece o scroll na página.

Esta div tem position: sticky;

A position static é a padrão dos elementos.

Esta div tem position: static;

As positions relative e absolute são usados para sobrepor dois elementos de uma página.

Esta div tem position: relative;
Esta div tem position: absolute;

O position fixed é muito usado para elementos como Navbar fixas no topo da página.
